# План Фазы 0–1 — karaokeshop.kz

> **Для исполнителя:** реализуй по задачам, отмечай шаги чекбоксами. Фаза 0 — research/setup (критерий приёмки вместо тестов). Фаза 1 — код (TDD где применимо: компоненты, SEO-утилиты).

**Goal:** Заложить SEO-фундамент (семантика, аналитика, данные, URL-карта) и поднять техкаркас (монорепо, дизайн-токены, UI-библиотека, роутинг, контент-модель, SEO-каркас) под полный ребилд.

**Architecture:** Монорепо Turborepo. Фронт — Next.js (App Router, SSR/SSG). Контент — Headless CMS (Sanity/Strapi, решается в Task 5). SEO — позвоночник: URL и IA проектируются от семантического ядра.

**Tech Stack:** Next.js 15, TypeScript strict, Turborepo, Tailwind, Style Dictionary, Storybook, Vitest, Lighthouse CI, Яндекс.Метрика + GA4.

**Опорные документы:** [стратегия](../strategy/2026-06-18-redesign-strategy.md) · [семантическое ядро](../strategy/semantic-core.md)
