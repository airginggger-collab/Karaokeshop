# design/ — дизайн-система и прототипы

**Что здесь есть:**
- ✅ [`design-system.md`](design-system.md) — **шпаргалка дизайн-системы «Сцена»**: все токены цвета (light + dark) с HEX, типографика (Unbounded + Manrope, шкала px), радиусы, отступы, сетка + готовые промпты для Google Stitch (тёмная тема). Под перенос в Figma/Stitch.
- 📐 [`visualization/`](visualization/) — **переносимый стандарт визуализации** (роли-токены `--color-*`, контраст WCAG AA, light/dark, семантика состояний, анти-ИИ) + справочник цветовых систем, готовая палитра MedLog и контраст-чекер [`check.mjs`](visualization/check.mjs). Универсальный и цвето-агностичный (цвета подбираются локально); раздаётся всем проектам. «Сцена» выше — конкретная реализация этого стандарта.

**Что здесь будет:**
- `prototypes/` — HTML-прототипы экранов (стартовые 7 собраны в чате: Главная, Каталог, Карточка, B2B «под ключ», Готовый комплект, мобайл, Checkout).
- `figma-link.md` — ссылка на рабочий Figma + правила связи токенов (Tokens Studio ↔ `packages/tokens`).

**Принцип:** единый источник токенов — `packages/tokens`. Figma и код синхронизируются от него, а не расходятся.

## Семь стартовых прототипов (перенести в Figma на Фазе 1, Task 12)
1. Главная — двухпутевой герой + доверие + рассрочка Kaspi.
2. B2B «Под ключ» — этапы + калькулятор по площади → заявка.
3. Каталог — фасетные фильтры + сетка карточек со сценарий-тегами.
4. Карточка системы — галерея, цена+рассрочка, комплект, двойной CTA, отзывы.
5. Мобильная главная — нижняя таб-навигация, тач-таргеты ≥48dp.
6. Checkout — Kaspi-рассрочка как приоритетный способ оплаты.
7. Готовый комплект по площади — состав + «под ключ включено» + тиры площадей.
