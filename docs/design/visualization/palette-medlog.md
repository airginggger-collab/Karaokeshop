# Палитра MedLog — под пациента

> Готовая палитра для MedLog (аудитория — пациенты). Выведена из подхода Stripe, ре-тюнинг: холодные нейтрали + тёплый тил-акцент + медицинская семантика.
> Собрана по [VISUALIZATION.md](VISUALIZATION.md); все пары текст/фон проверены на WCAG AA (см. [check.mjs](check.mjs)).

## Что и зачем (из Stripe → в MedLog)

- **Скелет Stripe:** глубокий текст, воздушный фон, один уверенный акцент, читаемость по WCAG.
- **Акцент blurple → тёплый тил** `#0d7a6e` — уводит от generic-SaaS-look, звучит как «забота», а не «платёжка».
- **Нейтрали — холодный cool-slate** (синий подтон), без зелени; тил живёт **только** в акценте (кнопка/логотип/ссылка).
- **+ Медицинская семантика** статусов (success/danger/warning/info) — отдельные роли, не бренд-акцент, каждая с иконкой (пройдёт greyscale-тест).
- **Крупнее и спокойнее** — базовый кегль ~17px, много воздуха (под пациента).

## Токены (light + dark)

```css
:root{
  /* Поверхности */
  --color-page:      #f3f5fa;   /* холодный воздушный фон */
  --color-bg:        #ffffff;   /* карточка приёма */
  --color-surface:   #e9edf4;   /* чипы / подложки */
  /* Текст */
  --color-fg:        #16202f;   /* cool-slate ink · 16.4:1 ✓ AAA */
  --color-muted-fg:  #57617a;   /* 6.2:1 ✓ */
  --color-border:    #d8dfea;   /* hairline */
  /* Бренд / действие */
  --color-primary:   #0d7a6e;   /* тил · белый текст 5.2:1 ✓ */
  --color-primary-fg:#ffffff;
  --color-link:      #0a6f64;   /* 6.1:1 ✓ */
  /* Семантика состояний (НЕ бренд-акцент) */
  --color-success:   #157f43;   /* принято */
  --color-danger:    #c62f28;   /* аллергия / тревога */
  --color-warning:   #f0a52a;   /* скоро / пропуск (тёмный текст) */
  --color-warning-fg:#3a2708;
  --color-info:      #1f6fb2;   /* инфо / заметка */
}
.dark{
  /* Поверхности — ре-тюнинг, не инверсия */
  --color-page:      #0c111b;   /* сине-графитовый, не чёрный */
  --color-bg:        #141b26;
  --color-surface:   #1e2735;
  /* Текст */
  --color-fg:        #eef1f6;   /* off-white · 15.3:1 ✓ */
  --color-muted-fg:  #97a2b4;   /* 6.7:1 ✓ */
  --color-border:    #283242;
  /* Бренд / действие */
  --color-primary:   #3ec8b5;   /* тил осветлён/десатурирован · тёмный текст 8.2:1 ✓ */
  --color-primary-fg:#04211d;
  --color-link:      #5fd6c4;   /* 9.8:1 ✓ */
  /* Семантика */
  --color-success:   #3fce87;
  --color-danger:    #ff7a7a;
  --color-warning:   #ecc14e;
  --color-warning-fg:#2a1e05;
  --color-info:      #5cb0ec;
}
```

## Контраст (проверено)

| Пара | Light | Dark |
|---|---|---|
| текст / карта | 16.4:1 ✓ | 15.3:1 ✓ |
| muted / карта | 6.2:1 ✓ | 6.7:1 ✓ |
| on-primary | 5.2:1 ✓ | 8.2:1 ✓ |
| ссылка / фон | 6.1:1 ✓ | 9.8:1 ✓ |

Все пары ≥ 4.5:1 (AA). Пересчитать: впиши пары в [check.mjs](check.mjs) → `node check.mjs`.

## Семантика статусов

| Статус | Роль | Иконка | Light |
|---|---|---|---|
| Принято | `success` | ✓ | `#157f43` |
| Скоро / пропуск | `warning` | ◔ | заливка `#f0a52a` + текст `#3a2708`; текст-на-белом `#a4661a` |
| Аллергия / тревога | `danger` | ✕ | `#c62f28` |
| Инфо / заметка | `info` | i | `#1f6fb2` |

## Развилки (открыты)

- **Вариант «врачи/профи»** — холоднее, плотнее, ближе к исходному Stripe (не делали).
- Если тил где-то занят — можно подобрать другой бренд-hue тем же методом.
