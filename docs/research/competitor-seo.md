## Где свободно — приоритеты (никто не закрывает)
1. **🎯 Сравнение брендов (F)** — `art-system` физически не может (только AST), остальные не делают. Лёгкий топ.
2. **🎯 По площади (H)** — лендинги «караоке для бара до 80 м²» + калькулятор. Прямой паттерн `karaoke-store.ru`, в KZ никто не использует.
3. **🎯 E-E-A-T контент (I)** — «как выбрать», «сколько стоит оснастить клуб», окупаемость. У всех тонко или пусто.
4. **🎯 KZ-язык** — все три только на русском. Hreflang RU↔KZ = вторая незанятая плоскость.
5. **🎯 Отзывы/кейсы с разметкой** — структурированные `Review`/`AggregateRating`, у конкурентов отсутствуют или не размечены → звёзды в сниппете дадут CTR-преимущество.

## Технические наблюдения
- **art-system.kz** — невалидный TLS (`unable to verify certificate`): прямой минус к доверию/ранжированию. Наша валидная HTTPS + быстрый SSR = преимущество.
- Все на готовых/старых платформах → вероятна слабая скорость. **Снять Core Web Vitals каждого через PageSpeed Insights** (добить в Task 2 при доступе) и заложить наше преимущество по CWV.
- Главный соперник по полноте — **karaoke-shop.kz**: бьём его не структурой (она у него есть), а скоростью, контентом (I), сравнением (F), площадью (H), отзывами и KZ-версией.
