## Часть 1 — Ре-колор тил → янтарь (только `packages/tokens/css/theme.css`)

Цвета правятся **только в `theme.css`** (единственный источник по CLAUDE.md; не дублировать в `globals.css`/`tokens.json`). Контраст AA перепроверить по каждому токену (в файле ратио задокументированы).

**Тёмная тема (`.dark`):**
| Токен | Было | Станет |
|---|---|---|
| `--color-primary` | `#4de3c8` | `#f59e0b` |
| `--color-primary-fg` | `#021a15` | `#1a1204` (тёмный на янтаре) |
| `--color-primary-soft` | `#143028` | `#2a1c08` |
| `--color-accent` | `#4de3a8` | `#f59e0b` |
| `--color-accent-fg` | `#d0fcea` | `#ffe9c2` |
| `--color-accent-soft` | `#112b20` | `#2a1c08` |
| `--night-accent` | `#2dd4bf` | `#f59e0b` |

**Светлая тема (`:root`) — вариант L2:**
- `--color-primary: #b45309` — глубокий янтарь для **текста/ссылок/цен** (`text-primary`), AA на белом (~5:1).
- `--color-primary-fg: #fff7ed`; `--color-primary-soft: #fef3c7` (тёплый золотой тинт).
- `--color-accent: #b45309`; `--color-accent-fg: #7c2d12`; `--color-accent-soft: #ffedd5`.
- `--warm-accent` (`#9a3412`) — уже оранжевый, оставить.
- **Золотые кнопки (L2):** primary-кнопки в светлой теме — **яркий янтарь `#f59e0b` с тёмным текстом `#3a2a06`**, а не глубокий `#b45309`. Т.к. `text-primary` (ссылки) должен быть тёмным для AA, а кнопки — яркими, вводим отдельный токен:
  - `--color-cta: #f59e0b` (обе темы), `--color-cta-fg: #3a2a06`.
  - На него переключаются **primary-кнопки**: компонент `@kk/ui` Button (primary-вариант) + инлайн hero/CTA-кнопки, использующие `--color-primary`/`--night-accent` как фон. Ссылки/бордеры/текст остаются на `--color-primary`.
  - В тёмной теме `--color-cta` = `#f59e0b` = как primary (визуально едино).

**Проверка на регресс:** пройтись по компонентам, где фон = акцент (кнопки, бейджи, активные чипы, шаги калькулятора, hero-CTA) — убедиться, что фон использует `--color-cta` (яркий), а текстовые акценты — `--color-primary` (AA). Тёмные секции (`--night-*`) уже дают тёмный фон — там янтарь яркий, ок.
