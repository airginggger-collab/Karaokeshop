## 2. Главный конкурент: karaoke-shop.kz

Самый опасный — украл название с дефисом, первым стоит в выдаче по «карaоке shop».

**Их профиль:**
- Официальный представитель AST в РК + Studio Evolution
- Физически сидят в **Астане** (Цeлиноград), Алматы — вторично
- Телефоны: +7 (777) 220 00 04 / +7 (707) 336 12 22
- Instagram: @karaoke_ast
- Есть страница «Стать дилером» → они сами работают через дилеров
- Trade-In — уникальная фишка, которой у нас нет
- Нет блога / нет контент-маркетинга (раздел блога пустой или мало статей)
- Дизайн: стандартный WordPress-шаблон, не адаптированный

**Уязвимости:**
1. Астана = не Алматы. По запросам «карaоке Алматы» мы географически ближе
2. Нет SEO-контента → они не ранжируются по информационным запросам
3. Нет калькулятора / онлайн-подбора → конверсия ниже
4. Нет современного UX → доверие ниже у молодёжи и бизнеса
5. Домен с дефисом хуже ранжируется и хуже запоминается

## 3. Ситуация с брендами и эксклюзивностью

> ⚠️ **Важно уточнить у заказчика:** являемся ли мы **эксклюзивным** дилером AST и SE в Алматы, или просто «авторизованным»? Это влияет на позиционирование.

- karaoke-shop.kz называют себя официальным представителем AST в РК
- KVD System называют себя официальным дистрибьютором AST
- 1music.kz также продаёт AST и SE
- Если несколько дилеров → конкуренция по цене не выиграть → надо выигрывать по сервису, контенту и доверию

## 4. Стратегия по 5 направлениям

### 4.1 SEO — занять информационный трафик (быстрый выигрыш)

Конкуренты почти не пишут контент. У нас уже 26 статей — это фора.

**Что делать:**
- Продолжить блог по кластерам: «карaоке для [тип заведения] Алматы», «сколько стоит [сценарий]», «как выбрать [компонент]»
- Добавить геостатьи: «карaоке для ресторана Алматы», «карaоке бар Алматы открыть», «карaоке Медеуский район»
- Ответные статьи на конкурентов: «AST vs Studio Evolution — что выбрать в 2026»
- Захватить брендовые запросы: «AST-250 купить Алматы», «Evobox цена Казахстан»
- Добавить `/sravnit` в sitemap (сейчас Google не видит эту страницу)
- Завести Google My Business с адресом → локальные запросы «карaоке рядом»

**Целевые запросы которых нет у конкурентов:**
```
карaоке под ключ Алматы
карaоке для кафе стоимость Казахстан
карaоке система для дома цена Алматы
AST-250 купить Алматы
Studio Evolution Evobox цена
монтаж карaоке Алматы
```

---

### 4.2 Репутация — E-E-A-T и доверие

У конкурентов: минимальные отзывы, нет кейсов, нет команды.

**Что делать:**
- Собрать реальные отзывы от клиентов → Google My Business (рейтинг в выдаче)
- Фото реальных установок → раздел /keysy с before/after
- Видео монтажей → YouTube канал (ссылка с сайта = сигнал E-E-A-T)
- Раздел «Команда» с реальными лицами и именами
- Сертификаты дилера от AST и Studio Evolution — отсканировать и выложить
- Логотипы реальных заведений, которые уже оснастили

---

### 4.3 Конверсия — выиграть сделку после перехода

Даже если пришли с конкурента — надо закрыть.

**Что делать:**
- Калькулятор сметы (уже есть) → добавить ROI-блок: «Окупится за X месяцев при N гостях»
- Квиз «Подберём за 2 минуты» (уже есть) → улучшить результат: показывать конкретную модель
- Сравнение с конкурентом: таблица «Мы vs karaoke-shop.kz» (осторожно, без имён — «Мы vs типичный магазин»)
- WhatsApp-бот: стартовое сообщение «Здравствуйте! Подберём систему — ответьте на 3 вопроса»
- Страница `/garantiya` — расписать гарантию детальнее конкурентов (у них 1 год, у AST 3 года)
- Trade-In? — изучить возможность ввести программу обмена (конкурент уже делает)

### 4.4 Бренд — стать «своими» в Алматы

karaoke-shop.kz — это Астана. Мы — Алматы.

**Что делать:**
- Везде: «Алматы», «доставка по Алматы за 1 день», «сервис-центр в Алматы»
- 2ГИС: добавить карточку с фото офиса → появимся в «карaоке магазин Алматы» на карте
- Партнёрство с ресторанами/кафе Алматы → взаимные упоминания
- Участие в событиях: Gastreet, ресторанные выставки, HoReCa KZ
- Instagram/TikTok: короткие видео монтажей, демо звука, «до и после»

## 6. Что НЕ делать

- **Не воевать ценой** — все продают те же AST и SE по схожим ценам. Цена не дифференциатор.
- **Не упоминать конкурентов по имени** на сайте — SEO-риск + юридический риск.
- **Не копировать Trade-In** без проработки логистики и юридики.
- **Не рекламироваться на Kaspi** — это другой сегмент (5–60k ₸), не наш.

---

## 7. Долгосрочная защита позиции

Когда достигнем топа — чтобы не сдвинули:

1. **Контент-рвы:** 50+ статей блога → переписать невозможно быстро
2. **Ссылочная масса:** партнёрства, упоминания в СМИ HoReCa KZ, YouTube
3. **Доменный авторитет:** `karaokeshop.kz` без дефиса — более ценный домен
4. **База клиентов:** email/WhatsApp рассылка по базе → повторные заказы, рекомендации
5. **Физическое присутствие Алматы:** шоурум/демо-зал → конкуренту из Астаны не скопировать
