# Бюджет проекта — karaokeshop.kz

> Контекст: **разработка самостоятельная (с AI-помощью), философия «максимум бесплатно».** При самостоятельной сборке деньгами запуск ≈ ₸0 — основная «стоимость» это время. Деньги появляются в трёх местах: регистрация юрлица для оплат, комиссии с реальных продаж, платные тарифы сервисов (только при перерастании free-лимитов).
> Цифры — ориентир на 2026, ставки/курсы меняются. Курс принят ~₸500 за $1. Комиссии Kaspi/эквайринга — **уточнять в договорах**.

## C. Оплата, Kaspi, юрлицо (стоимость бизнеса, не сайта)

> Без ИП/ТОО онлайн-оплату и Kaspi подключить нельзя. У заказчика юрлица пока нет — это отдельный шаг.

| Пункт | Стоимость |
|---|---|
| Регистрация **ИП** (онлайн, egov/Kaspi) | практически бесплатно, пара дней |
| Регистрация ТОО | небольшая госпошлина + время |
| Эквайринг (карты): ePay Halyk, CloudPayments, Kaspi Pay | **~2,5–3,5% с продажи**, подключение часто ₸0 |
| Kaspi Магазин (витрина) | комиссия зависит от категории — **уточнить в договоре** |
| Рассрочка 0-0-12 Kaspi | ⚠️ платит продавец, **ориентир ~10–12% от суммы** (с 1,5 млн ₸ ≈ 150–180k ₸). Точно — в договоре Kaspi |

«Бесплатная для покупателя рассрочка» — маркетинговый расход бизнеса, закладывается в маржу, на стоимость сборки не влияет.

## D. Рекомендация по оплате (поэтапно)

Чек дорогой (₸200k–2,5М) → сложный эквайринг на старте не нужен.

- **Этап 1 (запуск, инфра ₸0):** корзина + «Оформить» → **заявка + ссылка Kaspi / Kaspi Магазин / WhatsApp**. Свой эквайринг не нужен, платишь только комиссию Kaspi с реальных продаж. Совпадает с реальным поведением покупателей в KZ.
- **Этап 2 (пойдут заказы):** добавить card-checkout (ePay/CloudPayments) + Kaspi Pay API. Требует ИП + договор, ~2,5–3,5%.

Корзина и онлайн-оплата технически есть с первого дня; дорогую интеграцию эквайринга подключаем, только когда окупится.

## E. Итоговая смета

| Сценарий | Деньгами на старте | В месяц (первые месяцы) |
|---|---|---|
| **Максимум-бесплатно (выбран)** | **₸0–8 000** (домен) | **₸0–10 000** |
| + приём оплат | + ИП (~₸0) + комиссии **с продаж** (2,5–12%) | зависит от оборота |
| Комфортный (Vercel Pro + почта + CRM) | ~₸8 000 | ~₸25 000–40 000 |

**Вывод:** запуск реалистично почти за ₸0. Платное — по мере роста. Крупнейшая статья (комиссия рассрочки Kaspi) — расход бизнеса с продаж, не сайта.

## F. Хостинг — решение
**Держим ₸0:** старт на **Cloudflare Pages** (бесплатно, коммерция разрешена, безлимит-трафик, топ-CDN под скорость/SEO). Архитектура — переносимая: при упоре в ограничения адаптера уезжаем на Vercel Pro ($20/мес) в один шаг. Подробно и условия переезда — [ADR-0002: Хостинг](../adr/0002-hosting.md).
