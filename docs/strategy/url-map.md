## Технические правила
- **Canonical:** каждая посадочная — self-canonical. Параметры фильтров (`?brand=ast`) → canonical на чистый URL, кроме индексируемых комбинаций сценарий/площадь (для них — отдельные ЧПУ выше).
- **Hreflang:** каждая RU-страница `/x` ↔ KZ `/kk/x` (когда появится KZ-контент). Сейчас — зарезервировать структуру, не выпускать пустые KZ-страницы.
- **Sitemap:** разбить на `sitemap-pages`, `sitemap-products`, `sitemap-blog`.
- **301-редиректы** со старых URL текущего сайта — собрать карту перед запуском (Фаза 4).
- **Schema.org:** Product+Offer (товары), FAQPage (лендинги+блог), LocalBusiness (контакты), Review/AggregateRating (отзывы), BreadcrumbList (везде).
