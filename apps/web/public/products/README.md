# Фото товаров

Как добавить фото к товару — два способа.

## Способ 1 — локальный файл (рекомендуется для реальных фото)
1. Положи изображение сюда: `apps/web/public/products/<slug>.jpg`
   (slug — из товара в `apps/web/src/lib/site.ts`, напр. `rcf-evox-8.jpg`).
2. В товаре добавь поле:
   ```ts
   { slug: "rcf-evox-8", ..., image: "/products/rcf-evox-8.jpg" }
   ```
3. Готово — фото покажется в каталоге, на карточке и в сравнении.

Рекомендации: ~800×600 px и больше, JPG/PNG/WebP, до ~300 КБ (сожми, напр. squoosh.app).

## Способ 2 — внешний URL
Просто укажи ссылку в поле `image`:
```ts
{ slug: "...", ..., image: "https://.../foto.jpg" }
```

## Нет фото
Если поле `image` не задано — показывается иконка-заглушка. Можно добавлять фото по одному, постепенно.
